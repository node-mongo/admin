
## NodeMongoAdmin API

NodeMongoAdmin API is build with Node JS.
Sqlite2 is being used as the local data storage module:

## Features
- MongoDb installation status and overview.
- Databases overview and management.
- User(s) authentication and authorization.
- Collections and document object(s) management.
- Importing and exporting data.
- Processing overview.
- Administration tools.
- Application user management & MongoDb user management.
- MongoDb Atlas cloud database access and management

## Kick Off
To facilitate communication between the SPA and this API please start the Node V8 server.  
The simplest method:
$ cd api/src
$ node index.js
This will start the API service and the SPA can now make requests the API
